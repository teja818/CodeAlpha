# 📌 What is Phishing?

Phishing is a cyber attack where attackers pretend to be trusted organizations or individuals to steal:
- Passwords
- OTPs
- Bank details
- Personal information

Common phishing methods include:
- Fake emails
- Fake login pages
- SMS scams
- Social engineering attacks

This project includes a real-world PayPal phishing email example showing:

# 🔐 Cybersecurity Best Practices

✅ Verify sender email addresses  
✅ Avoid clicking suspicious links  
✅ Enable Two-Factor Authentication (2FA)  
✅ Never share OTPs or passwords  
✅ Use strong and unique passwords  
✅ Check HTTPS and domain spelling
